Назначение: единая инициализация и правила использования логгера.

Как инициализировать:

- В каждом модуле: `logger = get_logger(__name__)`.

Правила логирования:

- Логировать только на русском языке.
- Маскировать или не логировать секреты (ключи, токены, пароли).
- Использовать f-строки и явные имена переменных в сообщениях.
- Не использовать `print()` для трассировки — только логгер.

Уровни:

- `debug`, `info`, `warning`, `error`, `critical`, `exception` (в блоках try/except).

Пример:

```python
from modules.logger import get_logger

logger = get_logger(__name__)
try:
    do_work()
except Exception as e:
    logger.exception(f"Ошибка выполнения задачи: {e}")
```
